# Paper Figure Generation

This folder contains scripts used to generate paper-ready sensitivity figures for SER scheduling parameters.

## Average SER heatmaps

`average_ser_heatmaps.py` averages task-level `*-delta-pivot.csv` files generated by the RF/RN analysis script.

Each input CSV should contain percentage change in normalized AUC relative to the corresponding baseline.

Example:

```bash
python3 analysis/paper_figures/average_ser_heatmaps.py \
  --files task1-delta-pivot.csv task2-delta-pivot.csv task3-delta-pivot.csv \
  --title "ESER-SAC" \
  --name eser_sac_avg
